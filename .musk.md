# auto-musk 项目上下文

## 这是什么
auto-musk 是 auto-forge 的继任者,Rust 后端 + Vue 前端的 AI 编码助手。
后端在 `backend/`(Rust workspace),前端在 `gen/front/vue/`(a2vm 生成)。

## 技术栈
- 后端:Rust,tokio,axum,依赖 auto-ai(auto-ai-agent + auto-ai-client + ai-config)
- 前端:Vue3 + TypeScript + shadcn-vue(a2vm 从 .at 生成)
- LLM:经 auto-ai-daemon(aaid)调用,不直连 LLM API

## 常用命令
- `cd backend && cargo build` — 构建后端
- `cd backend && cargo test` — 跑测试
- `cd backend && cargo run -- run "task"` — 一次性任务
- `cd backend && cargo run -- chat` — 交互式会话
- `cd backend && cargo run -- serve` — HTTP API server(:8080)
- 前端:`cd gen/front/vue && npx vite build`(需先 `auto build` 重新生成)

## 代码约定
- 工具实现:`backend/crates/musk/src/tools.rs`,每个工具一个 struct + impl Tool
- 技能定义:`skills/<name>/SKILL.md`(frontmatter: name + description)
- API 端点:`backend/crates/musk/src/server.rs`,axum handler
- 活跃开发分支:`rust-impl`(main 保留给 Auto 复现)
